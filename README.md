### 🎯 [Talent Scout](https://github.com/Eslamelzamkan/Talent_scout)

**AI-Powered Hiring & Interview Agent System**

An end-to-end resume screening platform that ranks candidates against job descriptions using a two-stage AI pipeline:

🔹 **Stage 1** — Semantic ranking with fine-tuned SentenceTransformers (bi-encoder)
🔹 **Stage 2** — Structured analysis with deterministic scoring + optional LLM polish (Groq)
🔹 **NLP Extractors** — Skills matching, experience parsing, and entity recognition (spaCy NER)
🔹 **Full-Stack** — FastAPI backend · Streamlit UI · PostgreSQL persistence · JWT auth

> *Built to simulate a real recruiting workflow — from bulk resume upload to side-by-side candidate comparison.*
